前言

2.2 核心競爭優勢和護城河

金像電的核心競爭力在於其技術領先、客戶深度綁定、高效生產與策略擴產。

  1. 技術領先地位：專精於高層數、精密線路AI PCB（20層以上甚至30層以上），在高頻高速PCB技術上取得突破，能提供M8、M9等級CCL的高階需求。
  2. 深厚客戶關係：是美系雲端服務供應商 (CSP) 及AI ASIC主機板的主要供應商之一，與AWS、Google、Meta等客戶建立長期緊密合作關係，形成高轉換成本的護城河。
  3. 生產效率與規模經濟：台灣及蘇州工廠處於高稼動率運行，積極透過自動化和流程優化控制費用。泰國新廠的建設將進一步提升其全球AI供應鏈的領先規模。
  4. 優化產品組合：受惠於AI伺服器和800G交換器等高階產品，高毛利AI PCB佔營收比重不斷攀升，有效拉升整體毛利率。

3. 產業鏈上下游分析

電路板產業鏈複雜且環環相扣，金像電作為中游PCB製造商，其營運與上下游供應商及客戶關係緊密。

3.1 上游供應商分析

上游供應商主要提供製造PCB所需的銅箔基板 (CCL)、銅箔、玻纖布、光阻劑等關鍵材料。

3.1.1 主要供應商與供應鏈穩定性

  • **銅箔基板 (CCL)**：台光電、台燿、聯茂、南亞等台灣廠商在高階CCL市場扮演關鍵角色，尤其在AI伺服器應用方面。
  • **銅箔**：金居是高階HVLP4銅箔的領導供應商，榮科也是重要參與者。
  • **玻纖布**：台玻、富喬、建榮、南亞等是主要供應商。高階玻纖布 (如Low-CTE T-Glass、石英纖維布) 對AI伺服器至關重要，供應持續吃緊。
  • **光阻劑**：主要由日本廠商主導 (信越化學、東京應用化學、JSR) 及美國DuPont。
🔗

供應鏈穩定性： AI需求驅動高階材料 (如M9等級CCL、Q-glass石英布、HVLP4銅箔) 供應緊俏，交期拉長至數月。銅價飆漲也推升CCL成本。金像電透過高稼動率、產品組合優化及泰國廠擴建，強化供應鏈韌性。

3.1.2 議價能力

由於AI應用帶動高階材料需求強勁且供應吃緊，玻纖布、銅箔和CCL等上游材料供應商的議價能力顯著提升。例如，CCL廠商已因成本上漲而調漲產品價格。金像電在高階產品領域的領先地位，有助於其吸收部分成本壓力，但整體而言，上游議價能力增強對成本構成挑戰。

negotiating_hands

洞察： 上游高階材料供應商因技術稀缺與需求爆發，議價能力提升，對金像電的生產成本造成壓力。金像電需透過技術領先與客戶關係，將部分成本轉嫁至下游。

3.2 下游客戶分析

金像電客戶群明確聚焦於高階伺服器及網通設備領域，並與國際知名企業建立緊密合作關係。

3.2.1 主要客戶群與客戶集中度

  • **雲端服務供應商 (CSP)**：金像電是AWS (Amazon Web Services) 和Google等美系CSP客戶的ASIC主板主要供應商。在AWS Trainium 3的市佔率預計提升至60-70%，並預計在Google TPU領域取得至少20%的市佔率。
  • **網通設備商**：受惠於400G/800G交換器需求上升，並與1.6T網通客戶合作早期專案。
  • **伺服器為主要營收來源**：伺服器相關產品佔金像電營收約68%至69%，其中AI伺服器PCB因板層數增加而享有較高ASP和利潤。
🤝

客戶集中度： 金像電客戶群高度集中於少數國際大廠，雖然訂單穩定性高且能見度長 (已延伸至2026年)，但也意味著對主要客戶的依賴程度較高，需留意客戶需求變化或轉單風險。

3.2.2 銷售渠道與議價能力

金像電直接與大型CSP及網通設備商合作，透過提供高技術門檻、高層數、高頻高速板材的客製化解決方案，使其在市場上擁有較強的議價能力。其產品在AI伺服器供應鏈中扮演關鍵零組件角色，有助於維持高毛利率。

3.3 產業鏈關鍵環節和價值分配

金像電在中游PCB製造環節，尤其在高階伺服器與網通板領域，創造顯著價值。

  1. **高階技術領先**：能提供34-50層的高層數HLC板材，滿足M8、M9等級CCL的高階需求。
  2. **AI伺服器核心供應商**：作為ASIC PCB的主要供應商，提供從UBB到OAM的完整解決方案，強化競爭力。
  3. **網通設備升級**：受惠於400G/800G交換器與1.6T網通客戶的早期專案合作。
  4. **海外產能擴充**：泰國廠的建設有助於分散風險、提升接單彈性，並享受當地政府租稅優惠。
💰

價值分配： 金像電憑藉技術優勢和客戶深度綁定，使其在高價值產品市場擁有強大議價能力，產品平均售價（ASP）與利潤持續上升，毛利率達35.6%即為明證。

4. 產業趨勢與未來展望

全球PCB產業正迎接「黃金十年」，金像電在此波技術變革中掌握先機。

4.1 產業整體發展趨勢

  • **市場規模與成長率**：全球PCB市場預計到2029年將達965.7億美元，年複合成長率約4.87%至6.2%。台灣PCB產業鏈總產值預計2025年將成長至1.29兆新台幣，年增5.8%。
  • **技術變革**：
    • **ABF載板**：應用於AI晶片及5G基站晶片，朝更大尺寸、更多層數、更細線路發展，市場供不應求。
    • **高階HDI**：因應電子設備小型化與高效能，HDI應用於AI伺服器、低軌衛星、車用電子等。
    • **軟硬結合板**：結合硬板穩定性與軟板彈性，應用於物聯網、穿戴裝置、醫療、汽車等。
    • **高頻高速材料**：5G、AI/HPC伺服器、800G交換器等應用驅動對低介電常數和低損耗角正切材料的需求。
🚀

總體趨勢： AI應用主導PCB產業成長，技術變革推動高階材料與製程升級，地緣政治重塑供應鏈，加速東南亞擴廠。

4.2 新興技術和商業模式的影響

  • **AI驅動高階PCB**：AI伺服器與HPC對高層數、大尺寸、散熱、訊號完整性要求嚴苛，推動PCB製程朝半導體等級精密度發展。
  • **5G/6G高頻高速**：要求PCB具備超高頻、低損耗特性，促使高頻基材及先進封裝技術研發。
  • **電動車PCB價值提升**：電動車PCB用量及價值遠高於傳統車，BMS逐步採用軟板，ADAS推動HDI板需求。
  • **低軌衛星新藍海**：對PCB技術門檻極高，需要高階HDI製程、多種複合材料混壓、特殊雷射鑽孔技術。
  • **物聯網與邊緣運算**：推動PCB小型化、高整合，對HDI板和軟板需求提升，優化功耗、散熱與高頻傳輸性能。
  • **商業模式變革**：高附加價值服務、垂直整合、策略結盟、智慧工廠、綠色製造與利基市場深耕。

4.3 監管政策和宏觀經濟影響

  • **地緣政治風險**：美中貿易戰與科技戰驅動PCB產業供應鏈重組，加速台廠赴東南亞設廠。
  • **環保政策**：面臨嚴格的國際與國內環保法規，推動低碳轉型、製程節能、供應鏈減碳。
  • **匯率波動**：新台幣兌美元升值對出口導向的PCB產業構成壓力，可能侵蝕獲利。
  • **通膨與原物料成本**：全球通膨壓力及地緣政治事件可能導致銅、金等原物料價格上漲，增加生產成本。
  • **全球經濟景氣**：半導體產業榮枯直接影響PCB需求，AI、5G、電動車為主要動能。
🌍

影響評估： 宏觀經濟與地緣政治是PCB產業難以迴避的外部風險。金像電透過海外擴廠分散風險、提升韌性，並專注高階產品以應對成本壓力與匯率波動。綠色製造與智慧轉型則能強化其長期競爭力。

4.4 未來3-5年的成長機會和風險

4.4.1 成長機會

  1. **AI伺服器與高速運算 (HPC) 需求**：金像電在AI伺服器PCB和高速運算伺服器PCB領域具領先地位，特別是AWS Trainium 3、Google TPU等ASIC專案的訂單將持續增長，市佔率有望進一步提升至60-70%。
  2. **高階網通設備升級**：800G/1.6T交換器市場快速起飛，金像電在該領域的領先地位將顯著提升營收。
  3. **產品組合優化**：AI伺服器對PCB的層數、速度、散熱性能要求提升，金像電在高層數HLC板和M8、M9等級CCL材料的技術優勢，將推動ASP和毛利率持續增長。
  4. **產能擴張與全球佈局**：台灣、蘇州及泰國廠的產能擴張計畫將逐步貢獻營收，滿足市場需求並分散地緣政治風險。泰國廠一、二期將於2025年下半年和2026年下半年陸續投產。
  5. **新興應用領域**：儘管目前佔比較小，但車用電子、邊緣運算、低軌衛星等新興市場的發展，也為金像電帶來長期成長潛力。

4.4.2 風險

  1. **原物料價格波動與供應鏈風險**：銅價、電價等原材料上漲可能壓縮獲利空間。高階PCB所需材料供應有限，若出現短缺將影響生產。
  2. **市場競爭加劇**：台灣與中國PCB廠商在泰國等地擴產，可能導致市場價格競爭更趨激烈。金像電需持續保持技術領先與成本優勢。
  3. **AI伺服器市場需求不如預期**：儘管預期樂觀，但若AI伺服器市場增長放緩，或主要客戶訂單變化，可能影響營運表現。
  4. **地緣政治風險與全球經濟波動**：美中貿易戰、兩岸關係、全球經濟衰退等宏觀因素，可能影響供應鏈穩定性及終端電子產品需求。
  5. **技術快速迭代**：PCB技術不斷精進，若金像電無法跟上技術變革，可能影響競爭力。新技術研發投入大，也有失敗風險。
  6. **估值偏高疑慮**：部分PCB個股因AI題材漲幅驚人，若短期營收未能持續跟上市場預期，股價容易震盪。
  7. **新廠良率與學習曲線**：泰國新廠初期可能面臨良率爬坡與勞動力成本的挑戰，影響短期獲利。
